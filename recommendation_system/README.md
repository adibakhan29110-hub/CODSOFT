# Recommendation System (Content-Based)

This project recommends movies based on similarity of genre,
description, and title using TF-IDF and cosine similarity.

## How to Run

1. Create a virtual environment:
   python -m venv venv

2. Activate it:
   Windows: venv\Scripts\activate
   Mac/Linux: source venv/bin/activate

3. Install requirements:
   pip install -r requirements.txt

4. Run the program:
   python app.py

5. Enter any movie from the dataset:
   Example: Inception
